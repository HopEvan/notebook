# 继承
> extend

## ES5中的继承
```js
function Super() {}
Super.prototype.getNumber = function() {
  return 1
}

function Sub() {}
let s = new Sub()
Sub.prototype = Object.create(Super.prototype, {
  constructor: {
    value: Sub,
    enumerable: false,
    writable: true,
    configurable: true
  }
})
```
![alt](./imgs/extend-1.jpg)
以上继承实现思路就是将子类的原型设置为父类的原型
